# subjective_tool_data_fsearch

`subjective_tool_data_fsearch` is a Subjective data/search tool (PyQt-based UI) used for searching/indexing data within the Subjective ecosystem.

## Usage

Run from the Subjective monorepo (this repo is typically used as a submodule):

```bash
python brainboost_data_tools_search_index.py
```

## Notes

- This tool is designed to be used inside the main `Subjective` project (shared configuration/dependencies live there).
